# Karuna Vasu — Portfolio

Backend / platform engineer portfolio (Next.js 14 App Router, TypeScript, Tailwind CSS).

## Run locally

```bash
npm install
cp .env.example .env.local   # then fill in your values
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Folder structure

```text
karuna-portfolio/
├── app/                          # Next.js App Router (routes + UI)
│   ├── components/
│   │   ├── layout/               # Site chrome (Footer)
│   │   ├── sections/             # Homepage sections (Hero, About, …)
│   │   └── ui/                   # Reusable primitives (Nav, icons, shells)
│   ├── data/                     # Content + env-driven config
│   ├── lib/                      # Shared helpers (SEO / JSON-LD)
│   ├── projects/                 # Case-study routes
│   ├── resume/                   # Inline resume route
│   ├── robots.txt/               # Dynamic robots.txt
│   ├── sitemap.xml/              # Dynamic sitemap
│   ├── globals.css
│   ├── layout.tsx                # Root layout + metadata
│   └── page.tsx                  # Home page composition
├── public/                       # Static assets only
│   ├── profile.jpg
│   └── KarunaVasu_SoftwareEngineer_Resume.pdf
├── .env.example                  # Safe placeholders (committed)
├── .env.local                    # Real secrets (gitignored)
├── .vscode/                      # Shared editor excludes
├── next.config.js
├── package.json
├── tailwind.config.ts
└── tsconfig.json
```

### What you should NOT commit
| Path | Why |
|------|-----|
| `node_modules/` | Dependencies — reinstall with `npm install` |
| `.next/` | Build cache — regenerated by `npm run dev/build` |
| `.env.local` | Secrets / personal URLs |
| `next-env.d.ts` | Auto-generated by Next |

## Environment

1. Copy `.env.example` → `.env.local`
2. Fill contact, social, and product demo URLs
3. On Vercel, set the same `NEXT_PUBLIC_*` keys in Project Settings

## Deploy

Push to GitHub → connect the repo in Vercel. `NEXT_PUBLIC_SITE_URL` is optional; Vercel URLs are auto-detected.
